Missão 5 - A tag <!DOCTYPE html>

A tag <!DOCTYPE html> é uma das primeiras coisas que você verá ao começar a escrever um código HTML. Sem ela, o navegador pode não entender corretamente a página e pode haver erros na renderização.

Explicação sobre a tag <!DOCTYPE html>

A tag <!DOCTYPE html> serve para informar ao navegador qual versão do HTML está sendo usada. No caso do HTML5, essa tag é obrigatória para garantir que o documento seja interpretado corretamente como um documento HTML5.

Sem essa tag, o navegador pode entrar em um modo de compatibilidade (modo quirks), onde o layout da página pode não ser exibido corretamente, afetando a estrutura e o design da página. Quando a tag <!DOCTYPE html> é utilizada, o navegador sabe que a página deve ser interpretada como um documento HTML5 e renderiza o conteúdo conforme esperado.

Além disso, a tag <!DOCTYPE html> deve ser colocada no topo de qualquer página HTML, antes de qualquer outra tag, como <html> ou <head>.

Um exemplo de código com a tag <!DOCTYPE html>: 

<!DOCTYPE html>
<html>
    <head>
        <title>Página Exemplo</title>
    </head>
    <body>
        <h1>Bem-vindo à minha página</h1>
    </body>
</html>

No exemplo acima, a tag <!DOCTYPE html> define que estamos usando o HTML5, e ela deve sempre ser a primeira linha do código.
