Missão PY14 - Criar um código if, elif, else

Dica: Crie um código utilizando as estruturas de if, elif e else para verificar a idade de uma pessoa. O código deve exibir a mensagem "Você é maior de idade", se a idade for 18 ou mais. Se for exatamente 17, o código deve imprimir "Você tem 17 anos", e para idades menores deve imprimir "Você é menor de idade".

Exemplo de código (não copie, escreva o seu):

idade = 25
if idade >= 18:
print("Você é maior de idade")
elif idade == 17:
print("Você tem 17 anos")
else:
print("Você é menor de idade")

IMPORTANTE: Faça seu código baseado no exemplo, mas não copie diretamente. Use a estrutura do exemplo para definir a lógica e os prints, e modifique o valor da variável idade se desejar. A lógica deve ser a mesma, as mensagens números deve ser o mesmo.
