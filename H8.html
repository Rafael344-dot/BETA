Missão 8 - Criando um Botão

A tag <button> é usada para criar um botão em HTML. Você pode usar esta tag para ações como enviar formulários, abrir links ou executar funções de JavaScript.

Como Criar um Botão

O botão é uma das tags mais comuns e importantes do HTML. Ele permite que o usuário interaja com a página, realizando ações como enviar dados de um formulário ou ativar funções específicas.

A tag <button> não exige um fechamento especial, mas pode conter texto ou elementos dentro dela. Você pode até incluir ícones ou imagens, se necessário.

Exemplo de um botão simples:

Exemplo de Código:

<button>Clique Aqui</button>

No exemplo acima, criamos um botão simples que contém o texto "Clique Aqui". Você pode usar a tag <button> de diversas maneiras para realizar diferentes ações.

Agora, sua missão é escrever o código da tag <button> corretamente. Para esta missão, o código certo é o seguinte:

<button>[Escreva o texto aqui]</button>
