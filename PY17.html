Missão PY17 - Implementando a Lógica de If, Elif e Else

Explicação: Agora você vai avançar mais com as estruturas condicionais. Utilize if, elif e else para tomar decisões dentro do código, como verificar idades, permissões e mais. Ao completar esta missão, você estará mais perto de entender como criar lógica no Python!

Escreva um código com if, elif e else de acordo com o exemplo abaixo:

if idade >= 18: print("Você é maior de idade.") elif idade == 17: print("Você tem 17 anos.") else: print("Você é menor de idade.")
