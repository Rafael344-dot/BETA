Missão 7 - Funcionamento da Tag <img>

Agora que você já sabe como usar a tag <img>, vamos aprender mais sobre como ela funciona e como os atributos src e alt são essenciais para o seu uso.

Entendendo a Tag <img>

A tag <img> é usada para exibir imagens em uma página HTML. Ao contrário de muitas outras tags HTML, a tag <img> não possui uma tag de fechamento. Isso significa que ela não requer a tag </img>, como é o caso de outras tags.

A tag <img> funciona com dois atributos principais:

  • src: Este atributo especifica o caminho (URL) da imagem que você deseja exibir. Pode ser uma imagem local ou uma URL de uma imagem na internet.
  • alt: O atributo alt fornece um texto alternativo para a imagem. Esse texto é exibido caso a imagem não possa ser carregada, ou para leitores de tela usados por pessoas com deficiência visual.

Exemplo de Código:

<img src="URL_da_imagem" alt="Descrição da imagem">

No exemplo acima, a tag <img> exibe a imagem localizada no caminho especificado pelo atributo src, enquanto o atributo alt fornece uma descrição da imagem, útil para acessibilidade.
