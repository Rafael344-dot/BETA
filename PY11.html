Explicação Completa de If, Else e Elif no Python

Você já aprendeu a usar if e else no Python. Agora, vamos aprofundar mais e aprender sobre elif, que ajuda a fazer múltiplas verificações.

if: A estrutura if verifica uma condição. Se a condição for verdadeira, o código dentro do bloco será executado.

Exemplo:
if idade >= 18:
    print("Você é maior de idade.")

else: O else é o que será feito caso a condição do if não seja atendida. Ou seja, se a condição for falsa.

Exemplo:
else:
    print("Você é menor de idade.")

elif: O elif (abreviação de "else if") é usado quando você precisa de mais de uma verificação. Caso o if falhe, o Python tenta as condições do elif.

Exemplo:
elif idade == 16:
    print("Você tem 16 anos, que legal!")

Então, ao usar if, elif, e else, podemos criar estruturas de controle de fluxo mais robustas e inteligentes, que ajudam o código a tomar decisões com base nas condições fornecidas.

Abaixo está um exemplo completo de if, elif, e else:

idade = 18
if idade >= 18:
    print("Você é maior de idade.")
elif idade == 16:
    print("Você tem 16 anos!")
else:
    print("Você é menor de idade.")

Com isso, conseguimos tratar várias condições diferentes de forma mais limpa e organizada. O Python vai executar apenas a primeira condição verdadeira que encontrar.

Em resumo, você deve usar if para a condição principal, elif para verificações adicionais e else para a opção final, quando nenhuma das condições anteriores for verdadeira.
