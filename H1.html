Missão 1 - Introdução ao HTML

HTML (HyperText Markup Language) é usado para estruturar conteúdo na web. A tag <h1> é usada para criar um título principal em sua página.

Exemplo de Código

<!DOCTYPE html>
<html>
    <head>
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Aqui o usuário pode escrever o que quiser</h1>
    </body>
</html>

No exemplo acima, a tag <h1> cria um título principal. O texto entre as tags será exibido de forma destacada. Sua tarefa é reproduzir o código usando <h1>, mas com o texto que você desejar.

A tag <h1> é fundamental para definir títulos principais em páginas web. O que você colocar dentro dessa tag será destacado de maneira proeminente na página. Essa tag é de vital importância para a organização do conteúdo na web, pois o título principal geralmente é o primeiro elemento a ser visto pelos usuários.
