Missão 6 - Criando uma Imagem em HTML

Até agora, aprendemos a criar títulos com a tag <h1>, adicionar títulos de página com a tag <title>, e como a tag <!DOCTYPE html> é importante para garantir que o navegador entenda corretamente o nosso código.

Agora, vamos aprender a adicionar uma imagem em HTML!

Para adicionar uma imagem em HTML, usamos a tag <img>. Ela requer dois atributos essenciais: src e alt.

O atributo src especifica o caminho da imagem que você deseja mostrar, enquanto o atributo alt fornece uma descrição alternativa da imagem (para casos onde a imagem não pode ser carregada ou para usuários com deficiência visual).

Exemplo de Código:

<img src="URL_da_imagem" alt="Texto alternativo">

No exemplo acima, substitua URL_da_imagem pelo endereço da imagem que você deseja exibir e forneça uma descrição para o atributo alt.

Agora, sua tarefa é escrever o código HTML correto para exibir uma imagem!
