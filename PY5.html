Missão 5 - Variáveis e Print em Python

Em Python, as **variáveis** são usadas para armazenar informações que você pode usar e modificar ao longo do seu programa. Já o comando `print()` serve para exibir informações no console, permitindo que você veja os resultados do seu código.

Explicando as Variáveis e o comando `print()`:

Quando você cria uma variável, como por exemplo `nome = "João"`, você está dizendo ao Python que a palavra "João" está armazenada na variável chamada `nome`. Agora você pode usar essa variável no seu código em vez de escrever o valor diretamente. A função `print()` é muito usada para mostrar o valor de variáveis ou qualquer outra informação no console. 

# Exemplo:
nome = "João"      # Aqui, a variável 'nome' armazena o valor "João"
print(nome)         # O comando print vai mostrar o conteúdo da variável 'nome' no console

Então, quando você executa o código acima, o console vai exibir "João", porque foi o valor armazenado na variável `nome`.
