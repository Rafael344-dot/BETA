Missão 2 - Usando o print() em Python

O comando `print()` é utilizado para exibir informações na tela. Vamos aprender a usá-lo para exibir texto ou valores de variáveis. A tarefa é simples: digite o código abaixo usando o comando `print()`.

Exemplo de Código

# Usando o comando print() para exibir informações
print("Olá, Mundo!")

O código acima exibe a mensagem "Olá, Mundo!" na tela. Sua tarefa é escrever o código de forma similar, utilizando o `print()`.
