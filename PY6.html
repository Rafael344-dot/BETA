Missão 6 - Variáveis em Python

Agora que você aprendeu a usar variáveis e o comando print(), vamos praticar! O objetivo é criar duas variáveis, uma para o nome e outra para a idade, e imprimi-las usando print().

Exemplo de código:

nome = "João"
idade = 25
print(nome)
print(idade)

Agora, crie o seu próprio código. Defina uma variável para o nome e uma para a idade, e use o comando print() para exibir esses valores.
