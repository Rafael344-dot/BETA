Agora que já aprendemos sobre variáveis e print, vamos aprender sobre estruturas de controle no Python: if, else e endif. Essas estruturas nos permitem tomar decisões dentro do código.

if: A estrutura if é usada para executar um bloco de código apenas quando uma condição é verdadeira.

Exemplo:
if idade >= 18:
    print("Você é maior de idade.")

else: A estrutura else é usada quando a condição do if é falsa. Ela define o que deve ser feito se a condição não for atendida.

Exemplo:
else:
    print("Você é menor de idade.")

endif: O Python não usa a palavra-chave endif, mas ela pode ser mencionada como parte de uma explicação para fins de comparação com outras linguagens, como PHP ou Ruby, onde é usada para fechar a estrutura if.

Vamos ver um exemplo simples:

idade = 20
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

Essa estrutura permite que seu código tome decisões e execute blocos diferentes dependendo da condição fornecida.

A estrutura if e else são muito úteis para controle de fluxo, permitindo que seu código se adapte a diferentes situações.

O código abaixo mostra um exemplo básico de como usar o if e else para tomar decisões em seu programa:

idade = 16
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")
