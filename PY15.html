Missão PY15 - Explicação sobre If, Elif e Else

Explicação: Agora você entenderá como usar as estruturas de controle condicionais no Python. O if, elif e else são essenciais para controlar o fluxo de execução de um programa, verificando condições e tomando decisões baseadas nelas. Essas estruturas são amplamente utilizadas em programas reais.

Leia o código abaixo para ver como eles funcionam:

if idade >= 18: print("Você é maior de idade.") elif idade == 17: print("Você tem 17 anos.") else: print("Você é menor de idade.")
