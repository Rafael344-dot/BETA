Entendendo a estrutura condicional:

Você já aprendeu o básico sobre a estrutura de if, mas agora vamos aprofundar um pouco mais. A estrutura if permite que o programa execute um bloco de código apenas se uma condição for verdadeira. Caso contrário, o código dentro de um else ou elif será executado.

Vamos entender melhor cada parte da estrutura condicional:

1. if: O if é a palavra-chave usada para testar uma condição. Se a condição for verdadeira, o código dentro do bloco if será executado.

Exemplo:

if idade >= 18:
    print("Você é maior de idade!")

Isso significa que, se a variável idade for maior ou igual a 18, o programa imprimirá a mensagem "Você é maior de idade!".

2. elif: Se a condição do if não for verdadeira, você pode usar o elif (abreviação de "else if") para testar uma nova condição. Se o if não foi executado e a condição do elif for verdadeira, o código dentro do bloco elif será executado.

Exemplo:

elif idade == 18:
    print("Você tem 18 anos!")

Isso significa que, se a variável idade for exatamente 18, o programa imprimirá "Você tem 18 anos!".

3. else: Se nenhuma das condições anteriores forem verdadeiras, o código dentro do else será executado. O else serve para capturar todas as outras possibilidades.

Exemplo:

else:
    print("Você não tem 18 anos.")

Esse código será executado se a idade não for maior ou igual a 18 e também não for exatamente 18. Ou seja, ele é para todas as idades menores que 18.

Agora, depois de toda essa explicação detalhada, você está pronto para entender como usar if, elif e else em seus próprios códigos!
